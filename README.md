### Dataset Information

- `./raw_data/reimbursement_cpt_codes` - Data given to me by EDR, compiled from the data available from the CMS website for medicare physician fee schedules. This is not inflation adjusted.
- `./raw_data/inflation_table.csv` - Table created from data from the BLS's CPI inflation calculator. Shows conversion factors froms years 2000-2025.
- `./raw_data/medicare_part_b/utilizations` - Tables created from the Medicare Part B Fee-For-Service National Summary Data from 2000-2024.
- `./raw_data/provider_level/utilizations` - Tables created from the Medicare Physician & Other Practitioners Data from 2013-2023.
- `./raw_data/grouping_cpt_codes.csv` - Groups of CPT codes for downstream analysis.
